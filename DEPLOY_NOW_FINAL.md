## 🆘 IF BUILD FAILS

### Common Issues & Fixes:

#### 1. "Cannot find module 'figma:asset/...'"
**Cause**: Remaining Figma asset imports  
**Fix**: Check `/DEPLOYMENT_FIXES_APPLIED.md` for image replacement mapping

#### 2. TypeScript Errors
**Cause**: Type mismatches  
**Fix**: Run `npm run check` locally first

#### 3. "Module not found" Errors
**Cause**: Missing dependencies  
**Fix**: Run `npm install` then push again

#### 4. Build Timeout
**Cause**: Too many large files  
**Fix**: Already fixed with `.vercelignore`
