## 🛠️ HOW TO CREATE IMAGES

### **Option 1: Canva (Easiest - Free)**

1. **Go to Canva.com** (free account)

2. **Create Custom Size:**
   - Click "Create a design"
   - Custom dimensions: 1200 x 630 px

3. **Background:**
   - Add background color: #022c22
   - Or add gradient: #022c22 to #064e3b

4. **Add Logo:**
   - Upload your logo
   - Resize to 300 x 300 px
   - Position at top center

5. **Add Text:**
   - Title: "EMERALD CITY LIMOS"
     - Font: Montserrat Bold, 72px
     - Color: #d4af37 (gold)
   - Subtitle: "Seattle's Premier Black Car & Limo Service"
     - Font: Inter Regular, 40px
     - Color: #FFFFFF (white)
   - Services: "24/7 • Airport • Corporate • Weddings"
     - Font: Inter Light, 32px
     - Color: #FFFFFF

6. **Add Phone Number:**
   - Text: "📞 (206) 595-9675"
   - Font: Inter Medium, 36px
   - Color: #d4af37

7. **Add Accent Line:**
   - Insert shape → Line
   - Width: 600px, Height: 4px
   - Color: #d4af37
   - Position: Below subtitle

8. **Download:**
   - File type: JPG
   - Quality: Standard
   - Save as: `og-image.jpg`

9. **Optimize:**
   - Visit https://tinypng.com
   - Upload `og-image.jpg`
   - Download compressed version
   - Should be 100-300 KB

---

### **Option 2: Figma (Professional)**

1. **Create Frame:**
   - New file → Frame tool (F)
   - Width: 1200, Height: 630

2. **Background:**
   - Rectangle covering entire frame
   - Fill: Linear gradient
     - Top: #022c22
     - Bottom: #064e3b

3. **Logo:**
   - Place → Image → Upload logo
   - Resize: 300 x 300 px
   - Position: Center X, Y: 100

4. **Text Layers:**
   ```
   Title:
   - Text: "EMERALD CITY LIMOS"
   - Font: Inter Bold, 72px
   - Color: #d4af37
   - Position: Center X, Y: 280
   
   Subtitle:
   - Text: "Seattle's Premier Black Car & Limo Service"
   - Font: Inter Regular, 40px
   - Color: #FFFFFF
   - Position: Center X, Y: 360
   
   Line:
   - Rectangle: 600 x 4 px
   - Color: #d4af37
   - Position: Center X, Y: 420
   
   Services:
   - Text: "24/7 • Airport • Corporate • Weddings • Special Events"
   - Font: Inter Light, 32px
   - Color: #FFFFFF
   - Position: Center X, Y: 480
   
   Phone:
   - Text: "📞 (206) 595-9675"
   - Font: Inter Medium, 36px
   - Color: #d4af37
   - Position: Center X, Y: 560
   ```

5. **Export:**
   - Select frame
   - Export settings:
     - Format: JPG
     - Quality: 80%
   - Export as: `og-image.jpg`

---

### **Option 3: Photoshop (Advanced)**

1. **New Document:**
   - Width: 1200 px
   - Height: 630 px
   - Resolution: 72 dpi
   - Color Mode: RGB

2. **Background:**
   - Gradient tool
   - Colors: #022c22 (top) → #064e3b (bottom)

3. **Add Logo:**
   - File → Place Embedded
   - Resize to 300 x 300 px
   - Position at top center

4. **Add Text:**
   - Use Type Tool (T)
   - Follow text specifications above

5. **Save for Web:**
   - File → Export → Save for Web
   - Format: JPEG
   - Quality: 60-80
   - File size target: 100-300 KB

## 🎭 LOGO IMAGE

3. Center logo
4. Export as PNG with transparency

**Option 2: Canva**
1. Create design: 512 x 512 px
2. Transparent background
3. Add logo centered
4. Download as PNG

**Option 3: Remove.bg**
1. Visit https://remove.bg
2. Upload your logo
3. Download with transparent background
4. Resize to 512 x 512 px
